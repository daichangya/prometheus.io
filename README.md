## 特殊语法

如果想标注出需要读者特别注意的地方，可以使用以下语法：

```bash
> **[info] 注意**
>
> 冒号用来表示用户自定义的记录规则，不能在 exporter 中或监控对象直接暴露的指标中使用冒号来定义指标名称。
```
